# Introducción a la Estadística Descriptiva

## ¿Qué es la Estadística Descriptiva?

La **estadística descriptiva** es la rama de la estadística que se encarga de recopilar, organizar, resumir y presentar datos de manera informativa. A diferencia de la estadística inferencial, que busca hacer predicciones, la descriptiva se enfoca en describir lo que los datos muestran.

## Objetivos Principales

1. **Resumir grandes volúmenes de datos** en medidas significativas
2. **Visualizar patrones y tendencias** mediante gráficos
3. **Facilitar la comprensión** de conjuntos de datos complejos
4. **Detectar valores atípicos** y anomalías en los datos

## Componentes Fundamentales

### 1. Recolección de Datos
- Identificar la población o muestra
- Determinar variables de interés
- Establecer métodos de medición

### 2. Organización de Datos
- Tablas de frecuencias
- Distribuciones agrupadas
- Ordenamiento de datos

### 3. Resumen Numérico
- **Medidas de Tendencia Central**: describen el "centro" de los datos
  - Media (promedio)
  - Mediana (valor central)
  - Moda (valor más frecuente)

- **Medidas de Dispersión**: describen la variabilidad
  - Rango
  - Varianza
  - Desviación Estándar

### 4. Representación Gráfica
- Histogramas
- Diagramas de caja (boxplots)
- Gráficos de dispersión
- Gráficos de barras y pastel

## Aplicaciones en Ciencia de Datos

La estadística descriptiva es fundamental en ciencia de datos para:

- **Análisis Exploratorio de Datos (EDA)**: primer paso en cualquier proyecto
- **Limpieza de Datos**: identificar valores faltantes o erróneos
- **Feature Engineering**: crear nuevas variables basadas en resúmenes estadísticos
- **Comunicación de Resultados**: presentar hallazgos de forma clara

## Ejemplo Conceptual

Imagina que tienes las calificaciones de 100 estudiantes:

```
[85, 90, 78, 92, 88, 76, 95, 89, 84, 91, ...]
```

Con estadística descriptiva puedes:
- **Media**: Calcular el promedio (ej: 86.5)
- **Mediana**: Encontrar el valor central (ej: 87)
- **Desviación Estándar**: Medir qué tan dispersas están las notas (ej: 5.3)
- **Visualizar**: Crear un histograma para ver la distribución
